# HadaMackas

---

**Autoři:** Ondřej Kondom, Jiří Eidaman
**Engine:** Godot

### O čem to je?

---

HadaMackas je svižná 2D multiplayerová hra, ve které hráč ovládá hada pohybujícího se po mapě plné XP kuliček. Sbírej XP, zvětšuj své tělo, vyhýbej se ostatním hráčům a zkus je přelstít tak, aby narazili do tebe. Vyhrává ten, kdo přežije nejdéle a dosáhne největší velikosti. Styl a hratelnost jsou inspirovány jednoduchostí a návykovostí her jako slither.io.
